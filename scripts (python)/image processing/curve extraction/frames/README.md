# RAW FRAMES OF ALL TRIALS

## 1. Meaning of each repository name

e.g., `10H-10L` -> raw frame repository for trials with a pole height of 200mm and a strip length of 217mm.

See table below for pole height encoding:
- `10H` = 200mm
- `15H` = 300mm
- `20H` = 400mm
- `25H` = 500mm

See table below for strip length encoding:
- `10L` = 217mm 
- `15L` = 322mm  
- `20L` = 434mm 
- `25L` = 539mm
- `30L` = 651mm 
- `35L` = 756mm 

## 1. Meaning of each frame name in each repository
In each repository, there are the raw frames for each of the 27 trial groups (4 different strips x 7 different speeds).
e.g., `image_group_0_frame_193.0.jpg` -> a raw frame for strip n°1 at 150RPM.
